Where to catch it?

You can catch Phallaina at Ultima Thule. You have to teleport to Abode of the Ea. From there you have to fly to the island in the middle.

At this island it really doesn't matter where you try to catch this fish. You are able to fish all around here.

Ultima Thule

Ultima Thule is a location in Final Fantasy XIV: Endwalker and the final zone visited in the expansion story. Located at the farthest edge of the universe, it is a domain of pure dynamis, where emotions dictate reality, from being able to see and breathe to moving one place to the next.

When to catch it?

You can catch this fish from ten pm untill four am Eorzean time.

Any toher times catching this fish is impossible.

Bait to use:

You need to use in order to catch this fish stardust. Stardust you can get from the Scrip Exchange NPC (for example in Radz At Han)

Stardust cost about ten purple gatherer's scrip.

How to catch it?

First of all you will need a timer for ten seconds and not longer. For this you can just use a macro. If you have these than start fisshing, if you get a !! bite in ten seconds than hook it otherwhile just stop fishing because you gonna lose your bait! Normally the !! bite almost all the time is a Horizon Event.

Once you have this fish do the following skill combo in order to catch Phallaina:

Identical cast - Patience II - Powerfull Hookset - Mooch - Collect - Powerfull Hookset

And if you are lucky enough there you have your fish which you can turn in for a bunch of purple scrips.
